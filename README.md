# todo-app-handson

シンプルなタスク管理アプリケーション（ハンズオン）

## 概要
このリポジトリは、ブラウザの LocalStorage を使用したシンプルなタスク管理（ToDo）アプリです。

## 技術スタック
- HTML
- CSS
- JavaScript
- LocalStorage（ブラウザ）

## 特長
- タスクの追加、編集、削除
- タスクの完了マーク
- ブラウザの LocalStorage にタスクを永続化

## 使い方
1. リポジトリをクローンまたは ZIP をダウンロード
2. `index.html` をブラウザで開く

## 開発
開発や修正は単純な静的ファイル編集で行えます。ローカルで確認するだけならブラウザで開くだけで動作します。

## ライセンス
適宜プロジェクトのライセンスを設定してください。
